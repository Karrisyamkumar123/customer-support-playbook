USAGE GUIDELINES:
================

Logo Placement:
- Minimum clear space: 1x logo height on all sides
- Maximum size: 300px width for digital use
- Minimum size: 100px width for digital use
- Print minimum: 1 inch width

Color Usage:
- Primary: Use on light backgrounds
- White: Use on dark or colored backgrounds
- Never: Use on busy or patterned backgrounds
- Never: Stretch or distort the logo

Background Requirements:
- Light backgrounds: Use primary logo
- Dark backgrounds: Use white logo
- Colored backgrounds: Ensure sufficient contrast
- Transparent backgrounds: Available for all formats

Digital Usage:
- Websites: Use PNG or SVG formats
- Email signatures: Use PNG format, max 200px width
- Presentations: Use PNG format, maintain aspect ratio
- Social media: Use appropriate size for platform

Print Usage:
- Business cards: Use EPS or high-res PNG
- Letterheads: Use EPS format
- Brochures: Use EPS or AI format
- Large format: Use EPS or AI format

LOGO DO'S AND DON'TS:
=====================

DO:
- Use the logo at appropriate sizes
- Maintain clear space around the logo
- Use the correct color version for the background
- Keep the logo proportional
- Use vector formats when possible

DON'T:
- Stretch or distort the logo
- Change the logo colors
- Add effects or shadows to the logo
- Use the logo on busy backgrounds
- Make the logo too small to read
- Use outdated versions of the logo

ACCESSIBILITY:
==============

Contrast Requirements:
- Logo must have sufficient contrast with background
- Minimum contrast ratio: 4.5:1 for normal text
- Minimum contrast ratio: 3:1 for large text
- Test logo visibility on various backgrounds

Alternative Text:
- Always provide alt text for logo images
- Use descriptive alt text: "[Company Name] Logo"
- For decorative logos, use empty alt text: alt=""

RESPONSIVE DESIGN:
==================

Mobile Usage:
- Use icon version for small screens
- Minimum touch target: 44x44px
- Ensure logo is readable on mobile devices
- Test logo visibility on various screen sizes

Desktop Usage:
- Use full logo when space allows
- Maintain aspect ratio across screen sizes
- Ensure logo scales appropriately
- Test on different resolutions
